# results/

This directory is **generated by the pipeline** — it is intentionally empty in the
repo until you run the project.

Run either of these (Colab recommended, because the AlephBERT semantic model is
downloaded from HuggingFace, which some corporate networks block):

```bash
python src/run_all.py
```

or open `notebooks/NLP_Project.ipynb` in Google Colab and *Run all*.

After a run you will find here:

| File | Description |
|------|-------------|
| `metrics.csv` | Recall@k, MRR, nDCG@10, MAP for every retriever |
| `per_query.csv` | First-hit rank + reciprocal rank per (retriever, query) |
| `error_analysis.md` | ≥10 worked examples: semantic-wins / lexical-wins / both-fail |
| `figures/metrics_bar.png` | Recall@k & MRR per retriever |
| `figures/recall_curve.png` | Recall@k vs k |
| `figures/winloss_heatmap.png` | Per-query first-hit rank, lexical vs semantic |
| `figures/embedding_tsne.png` | t-SNE of AlephBERT passage embeddings by domain |
